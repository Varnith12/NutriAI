# NutriAI

NutriAI is an AI-powered calorie tracking and nutrition assistant designed to help users make healthier dietary decisions.

## Features
- AI-powered food analysis
- Calorie estimation
- Nutrition insights
- User-friendly dashboard
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Gemini AI API

## How to Run
1. Download the project files.
2. Open `nutriai.html` in a browser.
3. Enter a Gemini API key if required.
